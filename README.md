## 📖 소개

**오르락 내리락**은 한국 주식 시장을 시뮬레이션한 웹 기반 주식 게임입니다. 한국 거래소(KRX)에서 운영하는 어린이 주식 교육 게임에서 영감을 받아, 더 깊이 있고 전략적인 게임 경험을 제공합니다.

AI 투자자 4명과 실시간 순위 경쟁을 펼치고, 주식·경제 퀴즈를 풀어 보너스를 획득하세요!

## 🎮 게임 방법

```
1. 게임 시작 → 난이도 선택 → 플레이어 이름 입력
2. 매 턴 진행 흐름:
   📰 뉴스 카드 공개 → 📊 주가 변동 → 🧠 퀴즈 출현(40% 확률) → 🔄 매수/매도/대기
3. "대기" 버튼을 누르면 AI들이 매매하고 다음 턴으로 넘어감
4. 30턴마다 시즌 종료 → 리포트 카드 발급 → 다음 시즌으로 이어서 진행
```

## 🎮 게임 특징

### 📊 시장 사이클 시스템

4개 국면이 자동으로 순환합니다 (각 8~12턴, 랜덤):

| 국면 | 확률 | 특징 |
|------|------|------|
| 📈 상승장 | 호재 70% | 주가 상승폭이 큼 |
| ⚖️ 조정장 | 호악재 50:50 | 변동폭이 작음 |
| 📉 하락장 | 악재 70% | 주가 하락폭이 큼 |
| 🔄 반등장 | 호재 60% | 일부 종목 급등 |

### 🏢 8개 파로디 기업

| 종목 | 모델 | 섹터 | 특징 |
|------|------|------|------|
| 🚗 싱싱자동차 | Tesla | 자동차 | 경기 민감형 |
| 🔋 에너지K | LG에너지솔루션 | 2차전지 | 높은 변동성 |
| 📱 삼스전자 | 삼성전자 | 전자/반도체 | 안정적 |
| 💬 카카오트 | 카카오 | IT | 중간 변동성 |
| 🔍 네이버스 | NAVER | IT | 성장주 |
| 💾 SK하이닉스 | SK하이닉스 | 반도체 | 높은 변동성 |
| 💊 셀트리온 | 셀트리온 | 바이오 | 높은 변동성 |
| 🏦 KB금융 | KB금융 | 금융 | 방어주 |

### 🧠 퀴즈 모드

매 턴 **40% 확률**로 주식·경제 퀴즈 출제 (총 30문제):

- **주식 용어**: 상장, 매매체결, 배당, 코스피, 시가총액 등
- **경제 지표**: 금리, 환율, GDP, 물가지수 등
- **투자 심리**: 공포 탐욕 지수, 엔젤 투자자, 동적 자산배분 등
- 15초 타이머, 정답 시 보너스 자금, 2초 뒤 자동 닫힘

### 🤖 AI 투자자 순위표

4명의 AI 투자자와 매 턴 실시간 경쟁:

| AI | 전략 | 특징 |
|----|------|------|
| 🐜 개미투자자 | 랜덤 | 소액 거래, 감정적 매매 |
| 🏦 기관투자자 | 트렌드 추종 | 대량 거래, 추세 확인 후 진입 |
| 📊 가치투자자 | 가치 투자 | 하락 시 매수, 장기 보유 |
| 😎 나이스가이 | 균형 | 적절한 비율 관리 |

매 턴마다 순위 갱신, 🥇🥈🥉 메달 표시

### 📅 시즌제

- 1시즌 = **30턴** (게임 내 1년)
- 시즌 종료 시 **리포트 카드** 발급:
  - 총 자산 / 수익률 / 거래 횟수 / 최대 낙폭
  - 투자 스타일 분석 (단기투자자 / 가치투자자 / 모험가 등)
  - 시즌 등급: **S / A / B / C / D**
- 자산은 다음 시즌으로 누적 이어서 진행

### 🎲 시즌 이벤트

매 시즌 시작 시 랜덤 2~3개 이벤트 발생:

- 📊 배당락일 시즌 / 📈 실적 발표 시즌 / 🌍 해외 경제 위기
- 🏛️ 정부 경제 부양책 / 💰 특별 보너스 / 🔧 거래소 시스템 점검
- 🚀 신규 테마주 열풍

### 🎯 난이도

| 난이도 | 초기 자본 | 거래 수수료 | 특수 효과 |
|--------|----------|------------|----------|
| 일반 | ₩1억 | 0.015% | — |
| 어려움 | ₩5천만 | 0.3% | — |
| 전문가 | ₩3천만 | 0.5% | 변동성 +20% |

### 🏆 업적 시스템 (10종)

| 업적 | 조건 |
|------|------|
| 🎯 첫 거래 | 첫 번째 주식 거래 완료 |
| 💰 첫 수익 | 주식 매도로 수익 내기 |
| 📈 10% 수익률 | 총 자산 수익률 10% 달성 |
| 🚀 30% 수익률 | 총 자산 수익률 30% 달성 |
| 🌊 분산 투자 | 3개 이상 종목 보유 |
| 🔥 올인! | 현금 비율 10% 미만 |
| 🧠 퀴즈 달인 | 퀴즈 10문제 정답 |
| 🏆 AI 이기기 | 5턴 연속 1위 |
| 🐻 하락장 생존자 | 하락장에서 수익률 양수 |
| ⚡ 단기 매매왕 | 한 시즌 20회 이상 거래 |

### 🖥️ UI/UX

- 🌙 **다크 모드** 기본
- 🔴 빨간불 = 상승 / 🔵 파란불 = 하락 (한국 주식 컨벤션)
